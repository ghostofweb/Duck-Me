# 🚀 Quick Start Guide

## Install in 3 Steps

### 1️⃣ Get the Extension Files
You already have them! The extension is in this folder.

### 2️⃣ Load into Chrome
1. Open Chrome and go to: `chrome://extensions/`
2. Enable **Developer mode** (toggle in top-right)
3. Click **"Load unpacked"**
4. Select this `duckytabs` folder
5. Done! You'll see the 🦆 icon appear

### 3️⃣ Configure
1. Open your music tab (YouTube, Spotify, etc.)
2. Click the 🦆 DuckyTabs icon
3. Click **"Set Current Tab as Music"**
4. Adjust volumes if needed (defaults work great!)

## That's It! 

Now when you play learning content in other tabs, your music will automatically duck down. When you pause, it comes back up. Smooth and automatic! 🎉

## Pro Tips
- Set **Ducked Volume** to 15-25% for best focus
- Use **"Test Duck Effect"** to preview before committing
- You can change the music tab anytime
